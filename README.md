# googlesites-chromeapp
googlesites-chromeapp is a chrome app that redirects you to [https://sites.google.com](https://sites.google.com). 
how to install:
1. download a zip of this project
2. open [chrome://extesions](chrome://extesions)
3. check developer mode
4. click load unpacked extension
5. unzip the zip file you downloaded
6. select the unziped folder
7. go to chrome://apps and you will find the app there!
